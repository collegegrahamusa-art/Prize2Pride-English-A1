#### Activity 1: Input Activity

- **Learning Style Adaptation:** Use of diagrams, charts, and mind maps
- **Cognitive Level:** Remember
- **Description:** Students receive information about pop culture trends through visual modality
- **Time:** 10-15 minutes


#### Activity 2: Practice Activity

- **Learning Style Adaptation:** Color-coding and highlighting
- **Cognitive Level:** Understand
