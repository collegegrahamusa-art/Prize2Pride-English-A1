**Description:** Students write quick responses about using public transport  
**Materials:** Paper and pens or digital writing tools

### Main Activities


#### Activity 1: Input Activity

- **Learning Style Adaptation:** Reading comprehension activities
- **Cognitive Level:** Apply
- **Description:** Students receive information about using public transport through reading_writing modality
- **Time:** 10-15 minutes


#### Activity 2: Practice Activity

- **Learning Style Adaptation:** Writing assignments and journals
- **Cognitive Level:** Analyze
- **Description:** Students practice language related to using public transport using reading_writing approach
